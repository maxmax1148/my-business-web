# Netflix Business Website

เว็บไซต์จำลองของ **Netflix Business**  
จัดทำขึ้นสำหรับ **Assignment #1**  
เป็นเว็บไซต์ตัวอย่างเชิงธุรกิจของ Netflix  
นำเสนอข้อมูลเกี่ยวกับบริษัท บริการ แพ็กเกจ และช่องทางการติดต่อ

---

##  รายละเอียดเว็บไซต์

เว็บไซต์ประกอบด้วยหน้าเว็บหลักทั้งหมด **4 หน้า** ดังนี้

1. **หน้าแรก (Home)**  
   แสดงภาพรวมธุรกิจ Netflix แนะนำบริการหลัก และจุดเด่นของแพลตฟอร์ม

2. **เกี่ยวกับเรา (About)**  
   แสดงข้อมูลบริษัท ทีมงาน วิสัยทัศน์ และพันธกิจของ Netflix

3. **บริการ (Services)**  
   แสดงบริการต่าง ๆ เช่น  
   - บริการสตรีมมิ่ง  
   - บริการโฆษณา  
   - บริการวิเคราะห์ข้อมูล  
   พร้อมตารางเปรียบเทียบแพ็กเกจ

4. **ติดต่อเรา (Contact)**  
   ฟอร์มติดต่อพร้อม validation  
   ข้อมูลที่อยู่บริษัท และ Google Maps
