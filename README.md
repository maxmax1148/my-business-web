# Netflix  Website
เว็บไซต์ตัวอย่างสำหรับธุรกิจ Netflix ประกอบด้วยหน้าเว็บ 4 หน้า ได้แก่ หน้าแรก, เกี่ยวกับ, บริการ และติดต่อ
### 1. หน้าแรก (index.html)
- Header + Logo + Navigation
- Hero Section
- Featured Services
- Footer

### 2. หน้าเกี่ยวกับ (about.html)
- เรื่องราวบริษัท
- ทีมงาน (ใช้ `<figure>`)
- วิสัยทัศน์ / พันธกิจ
- Navigation กลับหน้าแรก

### 3. หน้าบริการ (services.html)
- รายการบริการ (ใช้ `<section>` + `<article>`)
- ตารางเปรียบเทียบแพ็กเกจ

### 4. หน้าติดต่อ (contact.html)
- Contact Form พร้อม Validation
- ที่อยู่บริษัท
- Google Maps Embed
